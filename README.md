Netflix Dataset Analyzer

Netflix Dataset Analyzer — настольное приложение для анализа датасета Netflix Movies and TV Shows. Программа позволяет загружать данные из CSV-файла, выполнять их предварительную обработку, анализировать распределение контента по странам, исследовать тренды выпуска фильмов и сериалов, строить графы связей актеров и режиссеров, а также получать рекомендации похожего контента на основе методов обработки текста и машинного обучения.

Для запуска приложения необходимо установить Anaconda, создать виртуальную среду проекта и установить зависимости. После этого достаточно активировать среду и выполнить команду:

python main.py

После запуска откроется графический интерфейс приложения. Пользователь может загрузить файл netflix_titles.csv, просматривать статистику по датасету, строить диаграммы и графики, анализировать связи между объектами и получать рекомендации по выбранным фильмам или сериалам. Все вычисления выполняются локально и не требуют подключения к Интернету.
